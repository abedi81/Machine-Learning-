# Machine Learning Projects

## Overview

This repository contains a collection of end-to-end machine learning projects covering both supervised and unsupervised learning.

The focus is on building complete pipelines from data preprocessing and feature engineering to model training, evaluation, and interpretation while connecting technical results to real-world insights.

The projects emphasize:
- Practical modeling decisions  
- Model comparison and validation  
- Interpretability  
- Business-oriented insights  

---

## Projects

### 1. Customer Churn Prediction (Supervised Learning)

A full machine learning pipeline to predict customer churn for a subscription-based platform.

**Key components:**
- Data cleaning and transformation (Log and Box-Cox)
- Feature engineering (engagement and behavioral metrics)
- Model comparison (Logistic Regression, SVM, Random Forest, XGBoost)
- Cross-validation and leakage testing
- Model interpretation (Feature importance, PDP, SHAP)

**Key insight:**
Customer engagement (watch time, login frequency) is the strongest predictor of churn, far outweighing demographic factors.

---

### 2. EV Charging Behavior Analysis (Unsupervised Learning)

Clustering EV charging sessions to identify distinct user behavior patterns.

**Key components:**
- Data preprocessing and feature scaling
- PCA for dimensionality reduction
- Clustering models (K-Means, GMM, DBSCAN, Hierarchical)
- Model evaluation and comparison

**Key insight:**
EV users naturally segment into four behavioral groups based on battery level and energy consumption, with heavy users driving peak demand and infrastructure stress.

---

### 3. Credit Card Customer Segmentation (Unsupervised Learning)

Customer segmentation based on financial and credit behavior.

**Key components:**
- Data cleaning and transformation (Log, Box-Cox)
- Feature scaling and preprocessing
- Clustering (K-Means, GMM, DBSCAN, Hierarchical)
- PCA for visualization and interpretation

**Key insight:**
Customers fall into distinct groups such as high-value spenders, credit-dependent users, inactive customers, and balanced users enabling targeted financial strategies.
